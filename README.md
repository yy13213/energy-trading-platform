# 能源交易平台

#### 介绍
蕴能汇是一款基于 FISCO BCOS 构建的新能源交易平台，通过区块链技术实现了电力能源交易链上可溯源、不可篡改等特性，支持电力能源交易、新能源设备交易等功能，提供安全、高效、透明的交易体验，为新能源行业的交易提供全新解决方案。
